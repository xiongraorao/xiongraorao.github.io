os包

通常我们可以采用os.system(cmd) 或者os.popen(cmd)来调用系统命令。

  • os.system(cmd): 返回值为0（成功执行）， 其实返回的是一个16位的二进制数，低8位为杀死所调用脚本的信号号码，高位为脚本的退出状态码

输出：
hello
3

  • os.popen(cmd): 该种方法是通过管道的方式来实现命令调用，函数返回一个file-like对象，里面的内容是脚本的标准输出，可以通过read()函数来接收

输出：
hello
